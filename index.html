Developer Information

This repository is maintained to provide information and SDKs for the Inventoryclerk.com Booking Calendar API.

The API is in two parts, the "coverage" API which is open to all, and the "booking" API which requires signed requests.

The SDKs are provided for your convenience but are not a requirement for working with Inventoryclerk.com.

Please note, the "booking" API requires you to have an account on Inventoryclerk.com. We provide "normal" accounts, whereby the client can query the API for their own booking information. We also provide "global" accounts with centralised invoicing but national scope. Please contact Inventoryclerk.com if you wish to be an account customer with us.
